# Vetores em JavaScript

Este repositório contém exercícios desenvolvidos em JavaScript com foco na utilização de vetores (arrays), estruturas de repetição e manipulação de dados armazenados em memória. As atividades foram realizadas sem a utilização de funções próprias de arrays, com o objetivo de fortalecer a compreensão da lógica de programação e do acesso direto aos elementos por meio de índices.

---

##  1 - Listagem de Nomes em Ordem Inversa 📄

O programa realiza a leitura de sete nomes informados pelo usuário e os armazena em um vetor. Após o preenchimento do vetor, os nomes são exibidos na ordem inversa daquela em que foram cadastrados, utilizando a navegação pelos índices do vetor de trás para frente.

---

##  2 - Cálculo de Média e Valores Acima da Média ➕

O programa lê oito números reais e os armazena em um vetor. Em seguida, calcula a média dos valores informados e percorre novamente o vetor para identificar e exibir apenas os números que estão acima da média calculada.

---

##  3 - Cadastro de Pessoas e Identificação de Menores de Idade 🥸

O programa realiza o cadastro de nove pessoas, armazenando os nomes e as idades em dois vetores relacionados. Ao final, são exibidos apenas os dados das pessoas menores de idade, mantendo a associação correta entre nome e idade por meio dos índices correspondentes.

---

##  4 - Interseção entre Vetores 📓

O programa realiza a leitura de dois vetores contendo cinco números inteiros cada. Após o preenchimento, é criado um terceiro vetor contendo apenas os valores presentes em ambos os vetores, sem repetição de elementos. O objetivo é identificar a interseção entre os conjuntos de dados armazenados.

---

## 💡 Conceitos Utilizados

- Vetores (Arrays)
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if`)
- Manipulação de índices
- Comparação de valores
- Armazenamento e recuperação de dados
- Lógica de programação

---

## 🎯 Objetivo do Projeto

Praticar a manipulação de vetores em JavaScript, reforçando conceitos de armazenamento de dados, percorrimento de estruturas, comparação de elementos e resolução de problemas utilizando lógica de programação sem o auxílio de métodos prontos para arrays.
